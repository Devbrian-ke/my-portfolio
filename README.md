## Getting Started

To run locally, simply clone the repo and open `index.html` in your browser — no build step needed.

```bash
git clone https://github.com/Devbrian-ke/my-portfolio.git
cd my-portfolio
# Open index.html in your browser
```

## Deployment

This site is deployed on **Vercel** with automatic deployments on every push to the `master` branch.

```bash
# To deploy updates
git add .
git commit -m "your update message"
git push
```

Vercel picks up the push and redeploys within 60 seconds.
